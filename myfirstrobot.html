Project Overview

MyFirstRobot was my very first robot. I decided to include it on this website as an honorable mention, not because it is technically advanced or especially interesting, but because it represents the starting point of my robotics journey.

It is a simple autonomous robot designed to move around randomly and react when it detects obstacles in front of it.

Image

How It Works

The robot moves in a random direction until the ultrasonic sensor detects an obstacle. When an obstacle is detected, the robot moves backward, rotates on itself, and then continues in a new direction.

The ultrasonic sensor is mounted on a servo motor, allowing it to measure distance in different directions instead of only looking straight ahead. This gives the robot a basic ability to scan its surroundings before choosing where to move next.

Main Components

  • Ultrasonic distance sensor
  • Servo motor for sensor rotation
  • DC motors for movement
  • Microcontroller-based control system
  • Basic obstacle avoidance logic

Why It Matters

Even though the robot is very simple, it was important because it helped me understand the basic logic of autonomous movement: reading sensor data, reacting to obstacles, controlling motors, and creating a first form of robot behavior.

For this reason, MyFirstRobot deserves a place here as the first step that led to more advanced robotics projects.
